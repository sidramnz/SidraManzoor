Monkey Business

Monkey Business makes use of a web application to send text or voice commands to control the movement of a physical model of a monkey.

Collaborated With: Billy Chen Ou Yang

My Role: Web Design | Technical Development

Toolkit: HTML | CSS | Javascipt | Arduino Yun

Inspiration

We chose to make a monkey model for this project after being inspired by Pocky, a snack company, that had recently released a seasonal flavor with a monkey as its mascot. By using the monkey as the center of our project, we worked to design an interactive product that could become a unique form of advertisement for Pocky.

Design & Technical Development

This project made use of an Arduino Yun that would get commands through the web application via Ajax. In the web application, based on what command was sent, the Arduino Yun would get a specific signal that would make the monkey move in specific ways.

For example if a user inputs the command “Wave” in the web application, the servos connected to the monkey’s arms would move at specific angles to make the monkey appear to wave.


Significance

Through this project, we were able to make a web application more interesting by adding in a physical element. We were also able to design a simple interactive product that would attract people towards it thereby helping to advertise a company’s product, in this case, Pocky’s snacks.
